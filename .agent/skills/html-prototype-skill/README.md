# HTML 原型图生成器 Skill

> 根据需求文档逐页生成线框风格 HTML 原型图，支持逐页审查和迭代修改

---

## 🎯 核心特性

- **线框风格**：灰色系 + 蓝色标注，专注于功能展示
- **功能标注**：每个功能点编号标注，底部详细说明
- **逐页生成**：一页一页输出，用户审查后再继续
- **迭代修改**：支持多轮修改，直到满意
- **浏览器预览**：生成后自动打开浏览器查看

## 🚀 使用方式

### 方式1：配合需求设计 Skill

```
1. 先使用 requirement-design-skill 生成需求文档
2. 然后说："根据需求文档生成原型图"
```

### 方式2：直接使用

```
请根据这个需求文档生成 HTML 原型图：[文件路径]
```

### 方式3：口述需求

```
帮我做一个登录页的原型图，需要包含：用户名、密码、登录按钮、第三方登录
```

## 🔄 工作流程

```
解析需求 → 提取页面列表 → 用户确认
                ↓
         ┌─ 生成第 1 页原型 ─┐
         │      ↓             │
         │  浏览器预览         │
         │      ↓             │
  迭代 ← │  用户反馈          │
  修改   │   ├─ 满意 → 下一页 │
         │   └─ 修改 → 迭代   │
         └────────────────────┘
                ↓
         全部完成，输出汇总
```

## 📝 标注体系

| 标注类型        | 样式                | 用途                 |
| --------------- | ------------------- | -------------------- |
| `[F1]` 功能编号 | 蓝色圆角标签        | 标注每个功能点       |
| 💡 交互说明     | 黄底气泡            | 说明交互行为         |
| 虚线功能区      | 虚线边框 + 蓝色标签 | 划分功能区域         |
| 底部标注说明    | 灰色底栏            | 所有功能点的详细说明 |
