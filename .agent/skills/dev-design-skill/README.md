# 前后端开发设计方案生成器 Skill

> 根据需求文档生成详细的前后端开发设计方案，支持功能点澄清、多方案推荐和技术选型理由

---

## 🎯 核心特性

- **逐模块设计**：按功能模块逐一输出前后端设计方案
- **主动澄清**：对不明确的功能点先提问再设计
- **多方案推荐**：关键技术决策提供多种方案对比
- **技术选型理由**：每个技术选择都说明为什么选它
- **前后端兼顾**：每个模块同时覆盖前端和后端设计
- **全局架构**：所有模块完成后输出整体架构设计

## 🚀 使用方式

### 方式1：配合需求文档 Skill

```
先生成需求文档，然后说："根据需求文档生成开发设计方案"
```

### 方式2：直接使用

```
请根据这个需求文档生成前后端开发设计方案：[文件路径]
```

---

## 🔄 工作流程

```
解析需求文档 → 提取功能模块 → 用户确认
               ↓
       ┌── 模块 1 ──┐
       │  澄清疑问   │
       │  前端设计   │
       │  后端设计   │
       │  多方案推荐 │ ← 迭代修改
       │  用户确认   │
       └─────────────┘
               ↓
       ┌── 模块 2 ──┐
       │  [同上流程] │
       └─────────────┘
               ↓
         全局架构设计
               ↓
         输出完整文档
               ↓
         后续操作（PDF / 原型图 / AI 开发）
```

---

## 📄 每个模块的设计产出

| 内容          | 说明                         |
| ------------- | ---------------------------- |
| 📌 功能概述   | 模块的核心功能和业务背景     |
| 🖥️ 前端设计   | 组件结构、状态管理、交互流程 |
| ⚙️ 后端设计   | 接口设计、数据模型、业务逻辑 |
| 🔗 前后端交互 | 接口调用时序和数据流转       |
| 🔀 多方案对比 | 关键技术点的多种方案和推荐   |
